# pdfs/

Source PDFs for the pipeline go here (`main.py` discovers everything
matching `*.pdf` in this directory). Gitignored on purpose — these are the
actual licensed product documentation, not ours to publish — so this
directory is otherwise empty in the public repo.

To run the pipeline yourself, place your own PDF(s) here, or use the
deployed app's Upload feature, then run `main.py` → `run_tail.py` per
`KT_Session_Document.docx`.
